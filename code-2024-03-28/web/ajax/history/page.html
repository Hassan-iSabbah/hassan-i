AJAX was invented in 1999 and BECAME POPULAR AROUND 2004.
AJAX STANDS FOR Async Javascript XML
The history of Ajax dates back to the early 2000s.


The term "Ajax" was coined by Jesse James Garrettin a blog post titled "Ajax: A New Approach to Web Applications," published in February 2005. However, the technologies and concepts that make up Ajax were in use before the term was coined.
  • 1996-1997:The XMLHttpRequest object, a crucial component of Ajax, was introduced by Microsoft in Internet Explorer 5. It allowed JavaScript to make asynchronous requests to the server.
  • 1998-2000:Various web technologies and techniques were developed, laying the foundation for Ajax. Technologies like JavaScript, Document Object Model (DOM), and XML were instrumental.
  • 2002:Microsoft's Outlook Web Access (OWA) introduced a new way of updating parts of a web page without reloading the entire page, which is a key principle of Ajax.
  • 2004:Google played a significant role in popularizing Ajax. Google started using Ajax techniques in Gmail, Google Maps, and other products, providing a smoother and more interactive user experience.
  • 2005:The term "Ajax" was coined by Jesse James Garrett in his blog post. This helped to bring together the various technologies and concepts under a common name.
  • 2006:The World Wide Web Consortium (W3C) started working on the XMLHttpRequest object specifications, standardizing the technology across different browsers.
  • 2007:jQuery, a popular JavaScript library that simplifies Ajax interactions among other things, was released. It significantly contributed to the widespread adoption of Ajax.
